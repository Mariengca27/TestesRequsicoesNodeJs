# TestesRequsicoesNodeJs

Código em linguagem JavaScript com o básico do básico de testes de requisições de comunicação:
        1º Com o cmd/shell;
        2º Com as respostas de requisições de http
        3º Com as respostas de requisições de várias URLs
Isso em paralelo com o NodeJs instalado na máquina para os testes feitos no shell, e verificação de eventos.
